# Portal Abonați — Demo

Prototip interactiv pentru un portal de abonați destinat unei companii de utilități (apă și canalizare). Single-file HTML, fără dependințe server.

![Demo](https://i.ibb.co/FbzQ8PHJ/demo-desktop-dashboard.jpg)

## Deschidere

```bash
open Portal_Abonati_Demo.html
```

Se deschide direct în browser. Fonturile se încarcă de pe Google Fonts.

## Ce conține

**Demo interactiv** (`Portal_Abonati_Demo.html`) — 11 pagini navigabile, date dummy JSON inline, filtre funcționale, toggle switches, notificări cu marcare citire.

**Documentație** (`Portal_Abonati_ApaCanal2000_Documentatie.md`) — specificații funcționale, arhitectură, fluxuri autentificare, wireframes, model de date, identitate vizuală.

## Pagini demo

- Panou principal (dashboard cu statistici live)
- Cont client
- Contracte și locuri de consum
- Facturi (cu filtrare pe status / contract)
- Istoric plăți
- Autocitire index (selector contor, formular, istoric)
- Sesizări (listă + formular depunere)
- Notificări
- Anunțuri
- Setări cont
- GDPR / Acord

## Stack

HTML + CSS + vanilla JS. Zero dependințe. Un singur fișier.
